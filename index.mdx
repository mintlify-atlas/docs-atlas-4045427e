fastarena is a high-performance arena allocator for Rust. It gives you O(1) allocation, zero-cost bulk reset, RAII transactions with auto-rollback, nested savepoints, and an arena-backed growable vector — all with zero external dependencies.

## Get started

<Steps>
  <Step title="Add fastarena to your project">
    Add the dependency to `Cargo.toml`:

    ```toml Cargo.toml
    [dependencies]
    fastarena = "0.1.3"
    ```
  </Step>
  <Step title="Create an arena and allocate">
    ```rust
    use fastarena::Arena;

    let mut arena = Arena::new();
    let x: &mut u64 = arena.alloc(42);
    let s: &str = arena.alloc_str("hello");
    let nums: &mut [u32] = arena.alloc_slice(0u32..8);

    // Zero-cost reset — pages stay warm, no OS calls
    arena.reset();
    ```
  </Step>
  <Step title="Use transactions for safe rollback">
    ```rust
    use fastarena::Arena;

    let mut arena = Arena::new();

    let result: Result<u32, &str> = arena.with_transaction(|txn| {
        txn.alloc_str("fastarena");
        let score = txn.alloc(100u32);
        Ok(*score)
    }); // Ok(100) — committed automatically
    ```
  </Step>
  <Step title="Explore the full API">
    Read the [Core Concepts](/concepts/arena-basics) guides and [API Reference](/api/arena) for all allocation methods, transaction controls, and ArenaVec operations.
  </Step>
</Steps>
